# Coffee Shop Sales - Data Analysis (Interactive Dashboard using MS Excel)

## Project Objective
To analyze coffee shop sales data and create an interactive Excel dashboard that tracks key KPIs such as total sales, footfall, and average order value.
This helps management understand performance trends and improve sales strategies.

## Dataset used
- <a href="https://github.com/vipin-s27/Coffee-Shop-Sales-Analysis/blob/main/Coffee_Shop_Sales_RawData.xlsx">Dataset</a> - Contains the raw sales data used to create the dashboard and analyze performance metrics.

## Questions (KPIs)
-	What is the total sales revenue?
-	What is the total footfall (number of customers)?
-	What is the average bill per person and average order per person?
-	What is the category-wise and size-wise sales distribution?
-	Which store location performs best in terms of revenue and footfall?
-	What are the top 5 best-selling products?
-	Which hours and days see the highest orders?
